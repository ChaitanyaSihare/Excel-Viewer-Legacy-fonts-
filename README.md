# Yatharoop

An open-source, privacy-first Excel viewer and Unicode conversion tool that runs entirely in your browser.

Features

- Open .xlsx files offline
- View multiple worksheets
- Convert legacy font text (KrutiDev, DevLys, and more)
- Export to Unicode Excel
- Export to PDF
- Export to HTML
- No file uploads
- No tracking
- Cross-platform (Web, Android, Desktop-ready)

Project Goals

Yatharoop is designed to help users work with legacy font spreadsheets while keeping their data on their own device. The project focuses on simplicity, privacy, and extensibility through an open-source Excel engine.
